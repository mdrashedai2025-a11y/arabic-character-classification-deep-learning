# 🧠 Arabic Character Classification using Deep Learning

End-to-end deep learning system for **Arabic character and numeral classification** using CNN, ResNet50, VGG16, and Vision Transformers (ViT).

---

## 🚀 Overview

This project focuses on building high-performance deep learning models for recognizing Arabic characters and numerals from images.

It explores multiple architectures and compares their performance to achieve optimal accuracy and efficiency.

---

## 🧠 Models Implemented

* Convolutional Neural Network (CNN)
* ResNet50 (Transfer Learning)
* VGG16 (Transfer Learning)
* Vision Transformer (ViT)

---

## 📊 Features

* Multi-model comparison
* Training & validation analysis
* Confusion matrix visualization
* Classification reports
* Model performance optimization
